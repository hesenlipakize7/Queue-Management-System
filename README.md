# Queue Management System

A simple Queue Management System developed with **Spring Boot**, **React**, and **PostgreSQL**.

## Features

- JWT Authentication
- Role-based Authorization (Admin / Operator)
- Operator Management
- Ticket Generation (Kiosk)
- Next Ticket
- Recall Ticket
- Skip Ticket
- Finish Ticket
- Transfer Ticket
- Queue Display Screen
- Queue Statistics

## Technologies

### Backend
- Java 21
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- PostgreSQL
- Lombok

### Frontend
- React
- Bootstrap
- JavaScript

## API Endpoints

- Login
- Operator Management
- Ticket Management
- Statistics
- Kiosk
- Display

## Run

Backend:

```bash
./gradlew bootRun
```

Frontend:

```bash
npm install
npm run dev
```
